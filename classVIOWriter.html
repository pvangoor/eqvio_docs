Detailed Description

A class to handle writing output for the VIO system.

The output is distributed over a number of files, which are organised in a given directory. The use of aofstream rather than ofstream means that the VIO system usually does not have to slow down or wait for the OS to write to files.

Create a number of files in the output directory to record VIO data.

Parameters
outputDirThe output directory where files should be stored.

This method creates the output directory if it does not already exist, and then creates output files for the VIO data to be written to. It also creates header rows in each of the files to make them human-readable.
