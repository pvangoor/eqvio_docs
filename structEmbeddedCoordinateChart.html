Detailed Description

template<int EDim, int Dim = Eigen::Dynamic>
struct EmbeddedCoordinateChart< EDim, Dim >

A Coordinate chart for an embedded manifold.

Given a manifold embedded in R^E, this struct is used to implement coordinate charts R^E -> R^m, where m is the dimension of the manifold itself.
