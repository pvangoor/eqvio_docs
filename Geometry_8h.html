Compute the numerical differential of a multivariate function.

Parameters
fThe function R^n -> R^m to be differentiated.
xThe point in R^n where the derivative is to be taken.
hThe step size used to calculate the differential. Optional.
Returns
The differential Df(x) as an m by n matrix.

This function computes the numerical differential of a function f at a point x by using the central difference approximation. By default, the step size is computed as the cube root of std::numeric_limits<double>::epsilon(). The optimal step size will depend on the exact function being differentiated, but this is a good starting point most of the time.
