Detailed Description

A line of csv values.

Reads a line as comma separated values and provides access to the data. By specialising operator<< and operator>>, many types of data can be easily read from and written to csv lines. Additionally, it is easy to read data in a csv that is listed in sequence. For example

myCSVLine >> myString >> myInt;

will read the first entry in the csv line as a string into myString, and then the next entry as an int into myInt.
